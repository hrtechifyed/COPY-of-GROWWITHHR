# PDF full-line justification update

The generated advisory PDF now treats full-width running copy as a continuous paragraph before line wrapping. Each non-final line is justified across the usable text width, and text moves to the next line only when the remaining words no longer fit logically on the current line.

Headings, labels, narrow table cells and deliberately centred copy retain their existing alignment.

The final **End of Report** page now places a mid-sized HRTechify logo in the centre above the closing title. The same layout applies to light, dark and dual-version report delivery.
